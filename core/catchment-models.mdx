## What is a Catchment Model

A key part of PICSy's functionality is the ability to switch quickly between different sets of metrics for a site, care home or planning application's location, and compare those metrics against your own or your organisation's benchmarks.

In PICSy these are called Catchment Models - a set of up to 8 metrics that you can optionally set target values for, and then see how any given location ranks against them. Anywhere you can see metrics you can select a target model for comparison and see the deltas between that location and whatever values you set.

## How To Use Catchment Models

Go to the [Catchment Models](https://picsy.uk/settings/catchment-models) page and edit or create a new catchment model by adding up to 8 metrics and (optionally) adding target values where relevant. You can drag the metrics to change their order. Give your model a name and save it, and it will be available to you and to other members of your organisation.

There is no limit to how many models you create - so if you need different models for different regions, or for different projects or sites, just create new ones. When you save a site, the catchment model you are using on that site will be stored with it so things stay organised.
