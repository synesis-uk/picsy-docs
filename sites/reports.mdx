---
title: 'Reports'
---

The Reports tab lets you generate, manage, and download saved site reports.

## Site reports header

<Frame>
  ![Reports Header 01](/images/reports-header-01.jpg)
</Frame>

At the top of the page, you can start report generation from the current saved site state.

## Report generation progress

<Frame>
  ![Reports Progress 01](/images/reports-progress-01.jpg)
</Frame>

After you start a run, the progress bar shows each stage from saving state through asset capture and PDF generation.

## Save settings before generation

<Frame>
  ![Reports Save Settings 01](/images/reports-save-settings-01.jpg)
</Frame>

If site settings have changed, you are prompted to confirm and save site details and run parameters before generating.

## Saved reports table

<Frame>
  ![Reports Table 01](/images/reports-table-01.jpg)
</Frame>

The reports table lists all saved report files for the site, including generated date, report type, filename, and row counts.

### Reports table actions

<Frame>
  ![Reports Actions 01](/images/reports-actions-01.jpg)
</Frame>

From each row, you can open, download, or delete the selected report.
