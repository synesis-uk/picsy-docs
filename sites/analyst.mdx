The Analyst tab provides an AI feasibility assessment for your selected site.

## AI feasibility assessment

<Frame>
  ![Analyst Overview 01](/images/analyst-overview-01.jpg)
</Frame>

This page combines report output and supporting map context in a single workspace.

## Generate report

<Frame>
  ![Analyst Generate 01](/images/analyst-generate-01.jpg)
</Frame>

Click **Generate report** to open the run form, then set the proposed beds, completion year, and grade before starting the assessment.

## Assessment report

<Frame>
  ![Analyst Report 01](/images/analyst-report-01.jpg)
</Frame>

When complete, the AI assessment appears as a structured report with section-by-section findings.

## Narrative map

<Frame>
  ![Analyst Narrative Map 01](/images/analyst-narrative-map-01.jpg)
</Frame>

The map panel follows the active report stage so you can review the relevant geography while reading each part of the assessment.
