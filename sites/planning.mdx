The Planning tab shows active and historical planning applications around your selected site.

## What you see on this page

1. **Site map and Planning layer**

The top of the page uses the shared site map layout with the Planning scene selected.

2. **Planning filters bar**

Under the map, you can filter planning applications before reviewing the table.

3. **Planning list table**

The main table shows planning applications for the selected site area.

4. **Metric controls (plan-dependent)**

If your plan includes metric view, you can switch between default and metric views and adjust comparison settings.

5. **Key competitor selection**

You can mark planning records as key competitors from the list.

6. **CSV export (admin plan)**

If your plan includes admin features, you can export the planning list as CSV.
